::: warning 人脸匹配
两张人脸进行 1：1 比对，得到人脸相似度
:::

## 说明

1. 人脸相似度比对：比对图片中两张人脸的相似度，并返回相似度分值。
2. 图片活体检测：分析单张图片中人像，判断图片中目标对象否为真人，确保比对效果真实可靠。
3. 支持多种图片类型：支持生活照、证件照、身份证芯片照等图片类型的人脸对比。

## 使用方式

::: details 在线 API
在线调用人脸对比接口，对比两张图片中人脸图片相似度，并返回评分，判断是否为同一个人。
:::
::: details 私有化部署
提供软件部署包私有化方案，可在本地服务器进行人脸 1：1 对比，适用于安全级别较高的内网环境。
:::
