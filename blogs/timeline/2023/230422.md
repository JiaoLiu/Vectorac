**OCR 文字识别引擎**正式上线！📝

## 能力概览

- ✅ **多语种支持**：中英文、简繁体、阿拉伯数字
- ✅ **多场景适配**：通用文档、表格、票据、证件
- ✅ **结构化输出**：保留版面信息，便于后处理
- ✅ **私有化部署**：支持本地化与端侧部署

## 简单接入

```js
const result = await ocr.recognize({
  image: '...',
  type: 'table'
})
console.log(result.text)
```

## 一些场景

| 场景 | 解决的问题 |
| --- | --- |
| 财务票据 | 自动录入票据信息，替代手工 |
| 合同文档 | 快速提取关键字段 |
| 名片识别 | 一键归档联系人 |
| 教育题库 | 批量识别题目入库 |

识别准确率与速度均处于行业领先水平，欢迎有需要的朋友试用。
