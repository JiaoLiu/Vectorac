**通用图表组件库 v1.0** 正式发布并开源！📊

## 特性一览

- 🎨 **丰富的图表类型**：折线、柱状、饼图、雷达、地图 ...
- 📦 **开箱即用**：按需引入，零配置上手
- 🛠 **高度可定制**：主题、交互、动画均可灵活扩展
- ⚡ **高性能**：基于 Canvas / SVG 双引擎

## 一行代码画个图

```js
import { LineChart } from '@vectorac/charts'

new LineChart('#app', {
  data,
  x: 'date',
  y: 'value'
})
```

## 为什么开源

> 我们坚信：**好的工具应该被更多人使用，也应该在更多人手中变得更好。**

项目已在 GitHub 开源，欢迎 ⭐️ Star、Fork、提 Issue 与 PR。让我们一起把它做得更好。
